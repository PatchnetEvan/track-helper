Privacy

MotoTrack is a session-only tool. It is designed to be forgettable — on purpose.

What we store

Nothing persistent. MotoTrack does not store any user data on our servers, because there is no server side to this app. The page is a static frontend.

What stays in your browser (the current session)

Anything you type — bike details, tire pressures, suspension clicks, lap times, notes — exists only in the current browser tab while it is open. It is held in memory by the page itself.

Refreshing the page, closing the tab, or navigating away clears the current session. We do not use cookies, IndexedDB, or any server-side storage.

Optional on-device history

If — and only if — you tap Save this session on the Review tab, that session is written to your browser's localStorage so you can review it later from the History tab. Nothing is written to storage automatically.

Saved sessions:

  • Stay on the device and browser where you saved them.
  • Are never sent over the network — there is no server to receive them.
  • Can be deleted one at a time, or all at once with Clear all history on the History tab.
  • Can also be cleared by your browser's normal "clear site data" controls.

Backups (export and import)

You can export your saved history to a JSON file and import it on another device. These are ordinary file operations that happen entirely on your machine — the file is saved to your device's downloads folder and read back locally. We do not see the file.

Accounts, tracking, analytics

There are no accounts, no logins, and no analytics or tracking scripts. We do not collect personal information and we do not build a profile of you.

Network requests

The app is a static page. It does not call any backend API to save your data. Your hosting provider (for example Cloudflare Pages) will see ordinary web traffic — the same as any other website visit — so it can deliver the page to you.

Advertising

Ads may be added to MotoTrack in the future. If that happens, this page will be updated to describe exactly what ad provider is used and what, if anything, it collects.

Changes to this policy

If the behavior of the app changes — for example, if any data begins being stored anywhere — this page will be updated first.
