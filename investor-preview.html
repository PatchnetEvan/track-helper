MotoTrack / Track Agent Pro

A local-first paddock notebook with an invite-only AI layer for structured session logging.

Email Evan Investor deck PDF available by email request only.

MotoTrack starts as a fast, local-first paddock notebook for riders who need to log sessions without accounts, cloud dependency, or friction.

Track Agent Pro adds an invite-only premium layer that helps riders convert messy track-session notes into structured review fields, including lap times, tire pressures, setup changes, and rider notes.

The first goal is not coaching or recommendations. The first goal is clean data capture. AI drafts, it never decides. The rider reviews, edits, or rejects structured fields before anything is saved.

This creates a foundation for future premium workflows around rider history, coach review, track-day organization, and provider integrations.

Problem

Riders lose valuable session data between sessions. Notes get scribbled on tape, buried in spreadsheets, or forgotten before the next session starts.

Solution

Fast logging plus AI-assisted structuring turns rough paddock notes into reviewable fields without removing rider confirmation.

Product Split

MotoTrack Log stays free, local-first, and account-free. Track Agent Pro is the premium invite-only workflow for structured session capture.

Current Status

Deployed prototype, invite-gated access, mock parser default, and AI provider path available but controlled.

Roadmap

Fast logging first. AI-assisted structuring second.

  1. Public interest capture
  2. Invited testers
  3. Controlled AI enablement
  4. Coach and provider workflows
